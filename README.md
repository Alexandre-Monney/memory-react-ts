## Installer le projet :

Clone App :  
`git clone git@github.com:Alexandre-Monney/memory-react-ts.git`

Install dependencies :  
`npm install`

Starting Dev Env :  
`npm run dev`

**visit** : http://127.0.0.1:5173/

Building Application :  
`npm run build`
